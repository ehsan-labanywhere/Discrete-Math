# 📚 Discrete Math Learning System

### Bridging Mathematics & Computer Science

An interactive **Streamlit**-based learning system for the textbook *Introduction to Discrete Structures*. Each chapter transforms static math concepts into an **active CS playground** with interactive visualizations, quizzes, and real-world applications.

---

## 🚀 Quick Start

```bash
# Install dependencies
pip install streamlit pandas numpy graphviz

# Run the home page
streamlit run home.py

# Or run any individual chapter
streamlit run chapter_1_logic.py
```

---

## 📖 Chapters

| Chapter | Topic | Key CS Bridges |
|---------|-------|---------------|
| **Ch 1** | Logic & Proofs | Logic gates, truth tables, data validation |
| **Ch 2** | Sets & Set Operations | Python sets, SQL JOINs, Venn diagrams |
| **Ch 3** | Proofs, Induction & Recursion | Loop invariants, recursion trees, call stacks |
| **Ch 4** | Functions | Hash functions, APIs, function pipelines |
| **Ch 5** | Counting & Combinatorics | Password security, birthday paradox, Pascal's Triangle |
| **Ch 6** | Relations | SQL tables, digraphs, topological sort, clustering |
| **Ch 7** | Graph Theory | Social networks, BFS/DFS, graph coloring |
| **Ch 8** | Trees | File systems, tree traversals, MST, Minimax AI |

---

## ✨ Features

- **Math ↔ CS Bridge**: Every concept shown side-by-side in LaTeX notation and CS code
- **Interactive Widgets**: Sliders, dropdowns, text inputs for dynamic exploration
- **Graphviz Visualizations**: Diagrams, graphs, trees, and arrow diagrams
- **Quick Check Quizzes**: Instant-feedback quizzes in every section
- **Real-World Applications**: From SQL queries to Tic-Tac-Toe AI

## 🛠️ Tech Stack

- [Streamlit](https://streamlit.io/) — Interactive web UI
- [Pandas](https://pandas.pydata.org/) — Data tables
- [NumPy](https://numpy.org/) — Matrix operations
- [Graphviz](https://graphviz.org/) — Graph visualizations
